Navigation help page

You can find your way around the pages quickly by using these navigational aids that are found in each Theme page.

Click on the red titles to learn more about each one.


Back button

  • Use your web browser's standard back button if you want to move back to a previous page.

Location Bar

  • The location bar is the quickest way of finding your way around. It displays the title of the current page you are viewing to the right hand side and the titles of the previous pages in the sequence as links. In the example above you could click on the Main Menu, the Unit List, and the Unit 3 Intro page.

  • The bar is only displayed at the top of the screen, so if you have scrolled down a page and it is no longer visible you may have to scroll back up to use it.

Continue / Next Link

  • If you are following the pages in a linear sequence (one after another), click on the link to view the next page in the suggested order.
