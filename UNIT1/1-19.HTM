This report, written as part of the Etgace project, illuminates key concepts of the project such as "active citizenship" and "governance." You will find how these terms have been defined and understood in each of the six national contexts. Each chapter provides a list of references to useful literature and websites within each of the national context, which you might find useful if you were to pursue more advanced reading on the key issues in active citizenship and governance.
